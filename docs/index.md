## Stack del curso

| Capa | Tecnología |
|------|-----------|
| Estado + DI | Riverpod |
| HTTP | dio (sin retrofit) |
| Modelos | freezed + json_serializable |
| Navegación | go_router |
| Seguridad | flutter_secure_storage |
| Backend | Cualquier REST API (JWT) |
| Testing | flutter_test + mocktail |
